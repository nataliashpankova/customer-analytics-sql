# 📊 Customer Segmentation Analysis using SQL

## 📌 Objective
Analyze customer behavior and purchasing patterns to identify high-value segments, improve retention, and guide business decisions.

## 📊 Dataset
Dataset for customer analytics including:
- Customer demographics (age, region)
- Purchasing behavior (avg_monthly_spend, purchase_frequency, avg_order_value)
- Engagement metrics (discount_usage_rate, support_interactions, return_rate, browsing_time_minutes)

**Source:** retail_customer_segmentation.csv (simulated / public dataset)

## 🔧 Tools
- SQL (MySQL / DBeaver)
- Optional: Python for data preprocessing

## 🔍 Analysis
- Customer segmentation (High / Medium / Low value)
- Top customers identification
- Behavioral analysis (spending vs purchase frequency, discount usage)
- Advanced analytics using CTEs and window functions

## 📈 Key Insights
- High-value customers contribute significantly more to overall revenue
- Majority of customers fall into medium-value segment
- Higher purchase frequency correlates with higher monthly spending

## 💡 Business Recommendations
- Target high-value customers with loyalty programs
- Increase engagement for low-value customers to improve retention
- Focus marketing efforts on medium-value segment to move them up
- Monitor discount usage to optimize promotions
